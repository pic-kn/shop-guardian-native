## 🛠️ TestFlight 配信フロー (Deployment Flow)
遠隔地のユーザーや実店舗のスタッフにアプリを配布するための手順です。

1.  **Apple Developer Program への登録**
    *   [Apple 公式サイト](https://developer.apple.com/programs/jp/)から登録（年間1.3万円程度）。これがないと TestFlight は使えません。
2.  **EAS CLI の準備**
    *   `npm install -g eas-cli` でツールをインストールし、`eas login` で Expo アカウントにログインします。
3.  **ビルド設定の作成**
    *   `eas build:configure` を実行し、iOS 用のビルド設定を生成します。
4.  **ビルドの実行とアップロード**
    *   `eas build --platform ios` を実行します。Apple の証明書作成なども自動で行われます。
5.  **TestFlight での招待**
    *   [App Store Connect](https://appstoreconnect.apple.com/) にログインし、アップロードされたビルドを選択。
    *   「テスター」タブから相手のメールアドレスを追加、または「パブリックリンク」を発行して共有します。
